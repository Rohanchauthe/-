# -
This project focuses on analyzing school students’ data and presenting key academic and attendance insights through an interactive dashboard built using Advanced Excel. The goal is to transform raw educational data into meaningful visual insights that can support data-driven decision-making for academic planning and performance evaluation.
📌 Project Overview

This project focuses on analyzing school students’ data and building an interactive dashboard using Advanced Excel.
The dashboard helps visualize academic performance, attendance patterns, and student distribution, making it easier to derive meaningful insights and support data-driven academic decisions.

🎯 Project Objectives

Analyze student performance across classes and subjects

Study attendance trends and their impact on academic results

Understand student distribution by class and gender

Build a clean, interactive, and user-friendly Excel dashboard

🗂 Dataset Description

The dataset contains school student information such as:

Class and section details

Gender

Subject-wise marks

Overall performance indicators

Attendance records and percentages

(Note: This is a sample/educational dataset used for analysis purposes.)

🔍 Analysis Performed

Class-wise and section-wise student count

Gender-wise distribution analysis

Subject-wise performance comparison

Attendance percentage analysis

Identification of high-performing and low-performing students

📊 Dashboard Features

Interactive Pivot Tables & Pivot Charts

Slicers and filters for class, gender, and subjects

KPI-style summaries for quick insights

Conditional formatting to highlight trends and performance gaps

Clean and structured dashboard layout

🛠 Tools & Techniques Used

Microsoft Excel (Advanced)

Pivot Tables & Pivot Charts

Slicers & Filters

Excel Functions:

IF

COUNTIFS

SUMIFS

AVERAGEIFS

XLOOKUP

Data Cleaning & Validation


📌 Conclusion

This project demonstrates how Advanced Excel can be effectively used for educational data analysis and reporting, converting raw student data into meaningful insights through interactive dashboards.
